Sterktes

Wetenschappelijke achtergrond

Ik heb een wetenschappelijk wiskundige opleiding gevolgd in het middelbaar, dus ik heb een goede ervaring met logische redeneringen en oplossingen af te leiden uit de beschrijvingen van problemen. Nu kan ik die basis effectief toepassen met mijn informatica competenties in mijn werk.

Dit kan ik aantonen met een het domein specifiek leerresultaat dat beschrijft dat de TI'er problemen kan analyseren, vertalen naar een IT-context en de informatiebehoeften van een organisatie gestructureerd weergeeft door gebruik te maken van analyse- en modelleringstechnieken. Enkele voorbeelden die hierbij aansluiten komen van het vak OOSA, Object Oriented System Analysis, waarbij ik problem statements, scope creep, constraints, statement of work beschrijf. Een ander vak is IT & Management, waarbij ik een business analysis doe en beschrijf hoe het probleem momenteel wordt aangepakt en hoe we willen dat de situatie er gaat uitzien.

Al deze zaken worden tentoongesteld door middel van context diagrammen, usecase diagrammen en descriptions, business modellen, activity diagrammen en workflow, conceptual en relational data model voor technical design, GANNT weergave van taken met een netwerk diagram voor planning voor het beste parallel pad, risisco analyse. Dit zijn zaken waarbij je logische redeneringen moet volgen om stapsgewijs van je probleem naar je oplossing te komen.

Projecten

Mazub alien game in JAVA

Heb op de KUL als project voor het vak Objectgericht Programmeren een heel Java spel van scratch gemaakt. Het gaat om een 2D platformer waarbij jij een alien bestuurt, met animaties, die dan tot de finish moet geraken. De Physics Engine werd van scratch gemaakt zonder enige tools of libraries, net als de collision interaction en de map generator. Dat koste me heel veel bloed zweet en tranen maar het eindresultaat was ongelooflijk mooi.

Huffman compressie algorithme in C++

Voor het vak Programmeren voor Multimedia uit de richting Toegepaste Informatica heb ik het compressie algorithme van Huffman, dat wiskundig gezien het meest efficiënt bewezen compressie algorithme is, nagemaakt in C++. Weer geen gebruik gemaakt van eender welke libraries, zelfs niet voor trees. De enige libraries die zijn gebruikt waren de standaard libraries van C++ en testing libraries. Persoonlijk vond ik C++ enorm leuk om aan te werken omdat het zo efficiënt is.

"Go" bord spel gemaakt in C#

Dan heb ik weer voor het vak Visuele Gebruikers Omgevingen uit Toegepaste Informatica een front end view gemaakt in C# en met WPF. De back end code, dus het model met de logica werd gegeven, en wij moesten een ViewModel en View maken en deze linken aan elkaar en aan het model. Dit was enorm tof omdat het een grafische applicatie is dus je kan er mooi mee interageren.
